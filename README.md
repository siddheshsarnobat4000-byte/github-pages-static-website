# Static Website Hosting with GitHub Pages

## Project Overview

This project demonstrates how to host a simple static website using GitHub Pages. The website is built using HTML and CSS and deployed directly from a GitHub repository.

## Objective

- Create a simple static website.
- Host the website using GitHub Pages.
- Learn Git and GitHub deployment workflow.
- Understand version control and website publishing.

## Tools Used

- HTML5
- CSS3
- Git
- GitHub
- GitHub Pages

## Project Structure

```text
project-folder/
│
├── index.html
├── style.css
└── README.md
```

## Features

- Simple and responsive webpage
- Custom styling using CSS
- Hosted on GitHub Pages
- Version controlled using Git

## Deployment Steps

1. Created `index.html` and `style.css`.
2. Initialized a Git repository.
3. Added and committed project files.
4. Created a GitHub repository.
5. Pushed the project to GitHub.
6. Enabled GitHub Pages from repository settings.
7. Selected the `main` branch and root folder.
8. Accessed the live website using the generated GitHub Pages URL.

## Git Commands Used

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin <repository-url>
git push -u origin main
```
